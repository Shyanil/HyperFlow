# **HyperFlow**  
**HyperFlow** is a scalar-valued autograd engine built from scratch in raw Python with extensive math operations, inspired by [karpathy/micrograd](https://github.com/karpathy/micrograd). It supports basic operations (`+`, `-`, `*`, `/`), and advanced functions like `sigmoid`, `leaky_relu`, `tanh`, `relu`, and `softmax`, along with proper backward propagation.

This project is a work in progress, with new features added regularly to make it more powerful and flexible.
