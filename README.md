# HyperFlow: A Scalar-Valued Autograd Engine in Python

**HyperFlow** is a lightweight, scalar-valued autograd engine, implemented from scratch in pure Python. It provides automatic differentiation and supports a wide range of mathematical operations, inspired by the simplicity of [karpathy/micrograd](https://github.com/karpathy/micrograd).

### Features:
- **Basic Operations:** Supports elementary arithmetic operations like addition (+), subtraction (-), multiplication (*), and division (/).
- **Advanced Functions:** Includes essential activation functions such as:
  - Sigmoid
  - Leaky ReLU
  - Tanh
  - ReLU
  - Softmax
- **Backward Propagation:** Proper implementation of backward propagation to compute gradients for all operations and functions.

### Key Highlights:
- **Efficient and Lightweight:** Built in raw Python, making it easy to understand, modify, and extend.
- **Automatic Differentiation:** The engine supports forward and backward passes, enabling the training of neural networks or optimization tasks.
- **Clear Design:** Focused on simplicity and clarity, ideal for educational purposes and experimentation.
